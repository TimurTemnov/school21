# Machine Learning Projects

Репозиторий с учебными проектами по машинному обучению (School 21).

## Содержание

1. **ML1 — Введение в машинное обучение**

   * Базовые понятия: supervised и unsupervised learning.
   * Классификация и регрессия.
   * Первые шаги построения модели (train/test split, метрики).

2. **ML2 — Линейная регрессия и регуляризация**

   * Аналитическое решение и градиентный спуск.
   * Ridge, Lasso, ElasticNet.
   * Bias-variance tradeoff, переобучение и недообучение.
   * Метрики качества (MAE, RMSE, R², MAPE).
   * Нормализация и полиномиальные признаки.

3. **ML3 — Оценка качества моделей**

   * Валидация: K-Fold, LOOCV, TimeSeriesSplit.
   * Подбор гиперпараметров: Grid Search, Randomized Search, Optuna.
   * Отбор признаков: Lasso, корреляция, permutation importance, SHAP.
   * Сравнение схем валидации, стабильность моделей.

4. **ML4 — Классификация**

   * Базовые алгоритмы классификации: Logistic Regression, Naive Bayes, KNN, SVM.
   * Метрики качества: Precision, Recall, F1, AUC-ROC, AUC-PR, Gini.
   * Построение confusion matrix и интерпретация ошибок.
   * Реализация логистической регрессии с нуля (SGD).
   * Работа с категориальными признаками (OneHotEncoder, CountEncoder).
   * Создание нелинейных признаков и улучшение качества модели.
   * Сравнение алгоритмов и выбор оптимальной модели.

5. **ML5 — Supervised Learning. Decision Trees и ансамбли**

   * Реализация с нуля:

     * `DecisionTreeClassifier21` / `DecisionTreeRegressor21` (критерии Gini и MSE)
     * `RandomForestClassifier21` (бутстрап, усреднение вероятностей)
     * `GDBTClassifier21` (градиентный бустинг по логистической потере)
     * `ExtraTreesClassifier21` (случайные пороги без бутстрапа)
   * Предобработка: `SimpleImputer`, `OneHotEncoder`, `CountEncoder`, `MissingIndicator`.
   * Сплит по времени (`PurchDate`) на train/val/test.
   * Оптимизация гиперпараметров через **Optuna**.
   * Сравнение с библиотеками:

     * `sklearn`: DecisionTree, RandomForest, GradientBoosting
     * `CatBoost`, `LightGBM`, `XGBoost (DART)`
   * Метрика: **Gini = 2*AUC − 1**.

6. **ML6 — Unsupervised Learning. Dimensionality Reduction**

   * Изучение методов снижения размерности для задач без целевой переменной.
   * Теоретические вопросы: проклятие размерности, различия между PCA и SVD, NMF и SVD, структура алгоритма Locally Linear Embedding.
   * Классификация с разреженными признаками:
     * Построение матрицы взаимодействий пользователь–книга (Sparse Matrix) на основе датасета книжных рекомендаций.
     * Предсказание возраста пользователя с помощью линейной регрессии и Random Forest.
     * Применение PCA и UMAP для сжатия признакового пространства и сравнение качества моделей и времени обучения.
   * Визуализация датасета MNIST:
     * Трансформация изображений цифр в двумерное пространство с помощью PCA, SVD, Randomized‑SVD, t‑SNE, UMAP и LLE.
     * Оценка разделимости классов с помощью предложенных метрик.
   * Сжатие изображений с использованием SVD:
     * Декомпозиция трёх случайных полутоновых изображений, выбор ранга для низкоранговой аппроксимации.
     * Анализ спектра сингулярных чисел и распределения объяснённой дисперсии.
   * Выделение фона видео:
     * Представление видеопоследовательности как матрицы, применение SVD для отделения фона с помощью низкорангового восстановления первого кадра.

7. **ML7 — Unsupervised Learning. Clustering**

   * Изучение алгоритмов кластеризации и их применение для генерации новых признаков.
   * Теоретический анализ методов инициализации центроидов K‑means (Forgy, Random Partition, собственный вариант).
   * Кластеризация на основе географических координат (долгота/широта) из датасета о недвижимости:
     * Реализация K‑means с нуля, расчёт Silhouette score и Distortion, визуализация кластеров.
     * Добавление меток кластеров как дополнительного признака в Lasso‑регрессию с масштабированием (MinMaxScaler).
     * Сравнение с реализацией K‑means из библиотеки sklearn.
   * Применение других алгоритмов кластеризации:
     * DBSCAN (подбор параметров для улучшения качества регрессии).
     * AgglomerativeClustering (с использованием имплементации из sklearn).
     * Gaussian Mixture (EM‑алгоритм).
   * Эксперименты с различными наборами признаков (включая bedrooms, bathrooms, interest_level) для поиска более эффективных кластеров.
   * Сравнение алгоритмов по сложности, скорости работы и влиянию на качество модели.
   * Сохранение решений в Jupyter Notebook с ответами на все обязательные вопросы.
